Meta Taggar

Vad är en <meta> tagg?

<meta> taggen innehåller ”metadata” om HTML dokumentet. Metadata visas aldrig på hemsidan utan det ligger bara i bakgrunden för att datorn skall läsa det.

Meta elementen används för att bland annat specificera sid-beskrivningar, nyckelord, författare av dokumentet, senast ändrad och mer.

<meta> taggar går alltid in i <head>. Metadata har alltid ett namn och värde, värdet anges genom ordet ”content”.

HTML5 introducerade en metod som lät utvecklarna bestämma hur hemsidan skulle visas på olika enheter med hjälp av <meta> taggen. Den kallas viewport.

<meta name="viewport" content="width=device-width, initial-scale=1.0">


”width=device-width” betyder att sidans bredd skall anpassas till enhetens bredd.

”Inital-scale=1.0” sätter zoomnings nivån när sidan laddas. 1.0=100% varken zoomas in eller ut.

XHTML

I HTML behöver man inte avsluta <meta> taggen.

I XHTML måste man avsluta </meta> taggen.


Exempel


<meta name="keywords" content="HTML, CSS, XHTML, meta">

Definerar nyckelord för sökmotorer.


<meta name="description" content="Projektarbete meta taggar">

Skapar en beskrivning för hemsidan.


<meta name="author" content="Gustaf Cavalli">

Anger författare för hemsidan.


<meta http-equiv="refresh" content="60">

Uppdaterar hemsidan varje minut.
